# claude-study

React + TypeScript + Vite で構築したWebアプリケーションです。

## アプリの概要

Claude を活用した学習用プロジェクトです。モダンなフロントエンド技術スタック（React 19 / TypeScript / Vite）をベースに構築されています。

## 主な機能

- React 19 によるコンポーネントベースのUI構築
- TypeScript による型安全な開発
- Vite による高速な開発サーバーとビルド
- ESLint によるコード品質の維持

## 起動方法

### 依存パッケージのインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

### ビルド結果のプレビュー

```bash
npm run preview
```
